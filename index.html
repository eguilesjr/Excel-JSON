Instructions
For Graboyes field staff:
1. When frames arrive on-site, enter the date in Shipment Received.
2. When installation starts, enter Install Started.
3. When fully done, enter Install Completed.
4. Add any field notes (damage, delays, punch list items).
5. Tap Sync to GitHub so the morning email picks up your updates.

Yellow-border fields are the only ones you need to edit. Everything else is reference data from Colfab.
