First-Time Setup Guide
1. Create Supabase database (5 min)
Go to supabase.com → New project → SQL Editor → paste and run the contents of supabase-setup.sql from the GitHub repo.

2. Get your credentials
Supabase dashboard → Settings → API → copy Project URL and anon/public key. Paste them above and tap Save & Connect.

3. Share with your team
Send the app URL and these same Supabase credentials to every PM. Each person enters them once in Settings. All updates sync in real time — no conflicts.

4. Set up the 6 AM email
In the GitHub repo: Settings → Secrets → Actions → add three secrets:
SUPABASE_URL · SUPABASE_ANON_KEY · GMAIL_APP_PASSWORD
